What is a Function?

Functions are the main "Building blocks" of the program.

They are "blocks" or small to large chunks of code that always stick together.

You can set them with a keyword, and use that keyword as many times as you like, instead of re-writing that entire code block over and over again.

For the most part, one of the main reasons for functions, is to avoid code duplication

What is a function parameter?

A parameter is a variable inside of the parentheses of a function.

What should a function declaration look like?

function name(parameters, delimited, by, comma) {

/*code*/

}
