What is a variable?

A variable is "Named Storage" for data.

It can be used to store data and be given any name that we give it. Variables allow you to setup your code.

A variable name should have a clean, obvious meaning, describing the data that it stores.

What is an example of a variable?

let message = 'Hello I am that data being stored in a variable. I am also called a string, because I am random text.'

("message" is the name of the variable)

alert(message); (The message text will now appear whenever you run this in the browser)

What are the 3 ways you can declare a variable?

  1. var (This is the old way of declaring, its not used much anymore)
  2. const (Semi short for constant. Use this if your variables will always be the same
  3. let (Use this instead of var. It was introduced to make it easier to see where variables live in our code)
